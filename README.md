# Panelia — Manga Online Platform

Веб-приложение для чтения манги, манхвы и маньхуа онлайн. Разработано на Next.js 14 с использованием App Router и TypeScript. Интерфейс полностью реализован по дизайну из Figma.

## О проекте

**Panelia** — онлайн-платформа для чтения манги, аналог Mangalib/Readmanga. Проект включает:

- Главную страницу с подборкой манги, закладками и последними обновлениями
- Каталог с фильтрацией по жанру, году, рейтингу и количеству глав
- Страницу манги с описанием, главами, рейтингом и комментариями
- Встроенный ридер глав
- Страницу профиля пользователя
- Модальное окно авторизации с OAuth (Telegram, Яндекс, VK, Google)

На данный момент — **полноценный UI-прототип** с моковыми данными, готовый к подключению бэкенда.

## Быстрый старт

```bash
# Перейти в папку с Next.js-приложением
cd panelia-next

# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

Приложение откроется на [http://localhost:3000](http://localhost:3000)

```bash

npm start
```

### Каталог — фильтры

Страница `/catalog` содержит 14+ хуков состояния и реализует:

- Поиск по названию
- Фильтрация по жанрам (мультиселект с поиском)
- Диапазон года выпуска
- Диапазон рейтинга
- Диапазон количества глав
- Несколько вариантов сортировки
- Пагинацию с умными эллипсами (`1, 2, 3 … 47, 48`)

## Компоненты

### `Header.tsx`

Навигационная панель с логотипом, поиском, переключателем темы и кнопкой входа. Управляет открытием `LoginModal`.

### `MangaGrid.tsx`

Сетка карточек манги. На мобильных устройствах отображает 2 карточки, на десктопе — все. Определяет размер экрана через `MediaQueryList`.

### `BookmarksCarousel.tsx`

Карусель на базе Swiper с 10 закладками и навигационными кнопками. Адаптируется под 6 брейкпоинтов.

| Ширина экрана | Слайдов видно |
|---------------|---------------|
| 0px           | 1.3           |
| 400px         | 1.8           |
| 540px         | 2.3           |
| 768px         | 3             |
| 1024px        | 4             |
| 1280px        | 5             |

### `UpdatesSection.tsx`

Список последних обновлений глав. Содержит:
- Переключатель «Только мои закладки»
- Кнопку «Показать ещё»

### `Footer.tsx`

Подвал с копирайтом, ссылками на юридические страницы, иконками VK/Telegram и брендингом «Made by».

### `LoginModal.tsx`

Модальное окно авторизации:
- Поля email и пароль
- OAuth-кнопки: Telegram, Яндекс, VK, Google
- Ссылка на регистрацию
- Закрытие по клику на оверлей или клавише Escape

```css

--font-body:        'Golos Text';

/* Компоновка */
--container-width:  1350px;
--section-gap:      100px;

/* Скругления */
```

### Шрифты

- **SONGER SemiExpanded** — заголовки, логотип

## Структура данных

Бэкенд ещё не подключён. Данные моковые, описаны TypeScript-интерфейсами прямо в компонентах:

```typescript
// Карточка манги
interface MangaCard {
  id: number;
  coverIdx: number;      // 1–12, маппится на cover_X.jpg
  title: string;
  rating?: number;
  genres?: string[];
}

// Глава
interface Chapter {
  volume: number;
  chapter: number;
  date: string;          // "DD.MM.YYYY"
  views: string;
  likes?: string;
}

// Комментарий
interface Comment {
  id: number;
  avatar: string;
  username: string;
  text: string;
  time: string;
  likes: number;
  dislikes: number;
}

// Детали манги
interface MangaDetail {
  title: string;
  genres: string[];
  likes: string;
  views: string;
  bookmarks: string;
  rating: number;
  ratingCount: string;
  description: string[];
  totalChapters: number;
  totalReviews: number;
}
```

## Адаптивность

Проект следует принципу **mobile-first**. Ключевые брейкпоинты:

- **320–539px** — мобильные
- **540–767px** — планшеты (портрет)
- **768–1023px** — планшеты (ландшафт)
- **1024–1279px** — ноутбуки
- **1280px+** — десктоп

Каталог на мобильных открывает фильтры как drawer-оверлей. На десктопе — боковая панель.
